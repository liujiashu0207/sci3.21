## 三、算法设计（已确认）

### 论文标题方向
面向移动机器人的残差驱动自适应 A* 路径规划算法

### 核心公式
```
f(n) = g(n) + α(n) × h_oct(n)

α(n) = 1 + β × (1 - ρ_local(n))

其中:
  h_oct(n) = max(dx,dy) + (√2-1)×min(dx,dy)    // Octile距离（无障碍基准）
  ρ_local(n) = 节点n周围(2r+1)×(2r+1)窗口的障碍率  // 积分图O(1)查询
  β ∈ [0, 0.5]                                   // 修正系数
  r = 5                                          // 窗口半径

物理含义:
  ρ_local 低（空旷）→ α 大 → 贪心加速搜索
  ρ_local 高（密集）→ α → 1.0 → 保守标准A*
  
设计来源: 残差学习思想
  h_real ≈ h_oct + Δh（障碍绕行残差）
  Δh 与 (1-ρ_local) 反相关 → α 与 (1-ρ_local) 正相关
```

### 三个创新点
1. **Octile 启发函数**替代欧氏距离（收紧基准）
2. **残差驱动的自适应权重** α(n) = 1 + β(1-ρ_local)（核心创新，文献无先例）
3. **两阶段路径平滑**（视线简化 + 拐角插值，含碰撞检测）

### 实验验证的效果（冒烟测试数据）
| 地图 | 障碍率 | 扩展节点 vs Octile A* | 路径最优? |
|------|--------|---------------------|----------|
| arena.map | 0.145 | **-50.0%** (β=0.3) | ✅ (β≤0.3) |
| Berlin_0_256 | 0.265 | **-56.2%** (β=0.3) | α-最优 |
| brc000d.map | 0.568 | **-38.7%** (β=0.3) | α-最优 |
| battleground | 0.648 | **-84.4%** (β=0.3) | α-最优 |

### SCI 阶段升级路径
```
中文核心: α(n) = 1 + β × (1 - ρ_local(n))          ← 单特征
SCI:     α(n) = 1 + β × sigmoid(多特征融合)          ← 多特征
         + B样条平滑（曲率连续）
         + Gazebo/ROS 实机验证
```
