## 算法组（6 组）

| # | 算法 | 启发函数 | 权重 | 平滑 |
|---|------|---------|------|------|
| G2 | A*(Euclidean) | 欧氏 | α=1.0 | 无 |
| G3 | A*(Octile) | Octile | α=1.0 | 无 |
| G4 | WA*(1.2) | Octile | α=1.2 固定 | 无 |
| G5 | Ours (RDA*) | Octile | α(n)=1+0.3×(1-ρ) | 两阶段 |
| G6 | NoAdaptive | Octile | α=1.0 | 两阶段 |
| G7 | NoSmooth | Octile | α(n)=1+0.3×(1-ρ) | 无 |

核心公式：
```
f(n) = g(n) + α(n) × h_oct(n)
α(n) = 1 + β × (1 − ρ_local(n))
ρ_local(n) = 节点n周围11×11窗口的障碍率（积分图O(1)查询）
```
